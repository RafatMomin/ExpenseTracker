# Expense Tracker in C

A simple console-based expense tracker application written in C.

## Features

- Add new expenses with a description and amount.
- List all recorded expenses.
- Display the total amount of all recorded expenses.

## How to Use

1. Download the project files and open a terminal.
2. Compile the program using a C compiler.
3. Run the compiled program.
4. Follow the on-screen menu:
    - Choose option 1 to add a new expense.
    - Choose option 2 to list all recorded expenses.
    - Choose option 3 to display the total of all expenses.
    - Choose option 4 to exit the program.
